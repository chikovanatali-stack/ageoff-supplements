# age:off supplements

Веб-приложение для каталога БАДов, подбора нутриционной поддержки и оформления заказа.

## Важно

- БАДы не являются лекарственными средствами.
- Приложение не ставит диагноз.
- Подбор носит справочный характер.
- Все формулировки должны соответствовать российскому законодательству о рекламе БАДов.
- Не использовать лечебные обещания.
- Не писать: лечит, вылечивает, избавляет от болезни, устраняет диагноз.

## Основные разделы приложения

1. Главная
2. Каталог БАДов
3. Карточка продукта
4. Анкета подбора
5. Результат подбора
6. Корзина
7. Оформление заказа
8. Личный кабинет
9. Моя схема поддержки

## Логика приложения

Пользователь проходит анкету по своим целям:

- кожа
- пищевой комфорт
- ЖКТ
- сон
- стресс
- энергия
- волосы
- базовые нутриенты

После анкеты приложение предлагает продукты для нутриционной поддержки.

## Юридическая формулировка

Информация в приложении носит справочный характер и не является медицинской рекомендацией. БАДы не являются лекарственными средствами и не предназначены для диагностики, лечения или профилактики заболеваний.
